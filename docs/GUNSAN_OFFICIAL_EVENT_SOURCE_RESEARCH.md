# 군산 공식 행사·축제 데이터 구조 점검

## 목적

이 문서는 군산.com 행사·축제 섹션에 사용할 데이터를 군산 공식 출처에서 가져올 수 있는지 점검한 기록입니다.

이번 작업에서는 홈 UI에 데이터를 반영하지 않고, 크롤링 구현도 하지 않습니다. 숨겨진 JSON/API 또는 안정적인 목록 구조가 있는지 확인하고, 향후 데이터 운영 방식을 정리합니다.

## 1. 군산문화관광 축제/행사 정보

- 공식 출처명: 군산문화관광 축제/행사 정보
- URL: https://www.gunsan.go.kr/tour/m2101
- 데이터 제공 방식: HTML 게시판 목록 + 상세 페이지
- 공개 JSON/API 확인 여부: 확인 안 됨

### 확인한 구조

목록 페이지에서 확인 가능한 정보:

- 총 게시물 수
- 행사명
- 장소 또는 주소
- 전화번호
- 상세 페이지 링크

확인된 목록 예:

- `2025군산짬뽕페스티벌`
- `2025 군산시간여행축제`
- `2025 실버마이크 전라·제주권`
- `군산 수제맥주&블루스 페스티벌`
- `8월의 크리스마스 골목길 축제`
- `군산새만금 국제마라톤대회`
- `군산꽁당보리축제`

상세 페이지 URL 구조:

```txt
/tour/m2101/view/{게시글ID}
```

상세 페이지에서 확인 가능한 정보:

- 행사명
- 이미지
- 장소
- 축제기간
- 기본정보 본문
- 일부 외부/공식 홈페이지 링크

예시 상세 페이지:

```txt
https://www.gunsan.go.kr/tour/m2101/view/5235377
https://www.gunsan.go.kr/tour/m2101/view/306508
```

### 필드별 추출 가능성

| 필드 | 가능 여부 | 비고 |
| --- | --- | --- |
| 행사명 | 가능 | 목록/상세에서 확인 가능 |
| 기간 | 가능 | 상세 페이지의 `축제기간` 영역에서 확인 가능 |
| 장소 | 가능 | 목록과 상세에서 확인 가능 |
| 이미지 | 가능 | 상세 페이지에 이미지가 있음 |
| 상세 링크 | 가능 | `/tour/m2101/view/{id}` 구조 |
| 전화번호 | 가능 | 목록에 일부 표시 |
| 실시간 상태 | 불가 | 진행 중/오늘 행사로 표현하면 안 됨 |

### 한계

- 공개 JSON/API가 확인되지 않았습니다.
- 직접 HTML 파싱이 필요할 수 있으나, 이번 단계에서는 크롤링을 구현하지 않습니다.
- 목록/상세 HTML 구조가 바뀌면 자동 수집이 깨질 수 있습니다.
- 이미지 사용권한과 재사용 조건을 별도로 확인해야 합니다.
- 날짜와 장소는 공식 출처 기준으로 표시할 수 있지만, “오늘 진행 중” 같은 상태 계산은 별도 검증이 필요합니다.

### 추천 사용 방식

1. 운영자가 군산문화관광 페이지를 보고 행사 seed를 수동 등록합니다.
2. 각 행사 카드에는 공식 출처 링크를 저장합니다.
3. 자동 수집은 관리자 입력 구조가 생긴 뒤 별도 검토합니다.
4. 자동화가 필요하면 크롤링이 아니라 공식 API 또는 제공 협의 가능성을 먼저 확인합니다.

## 2. 군산시청 시정소식/공지/행사 관련 페이지

- 공식 출처명: 군산시청 공식 도메인 내 공지/행사성 게시판
- 데이터 제공 방식: HTML 게시판 가능성
- 공개 JSON/API 확인 여부: 이번 점검에서 안정적인 행사 API를 확인하지 못함

### 확인 방향

군산시청 공식 도메인에는 공지, 시정소식, 문화관광 관련 게시판이 존재할 수 있습니다. 다만 이번 작업 범위에서는 `행사·축제 카드`에 바로 연결할 수 있는 안정적인 JSON/API를 확인하지 못했습니다.

### 추천 사용 방식

- 행사 후보를 발견하는 보조 출처로 둡니다.
- 홈 행사 카드 자동화의 1순위로 두지 않습니다.
- 향후 관리자 입력 화면에서 공식 링크를 함께 저장하는 방식이 적합합니다.

## 3. 군산시간여행축제 공식 사이트

- 공식 출처명: 군산시간여행축제 공식 사이트
- URL: https://festival.gunsan.go.kr
- 데이터 제공 방식: HTML 페이지 + 게시판
- 공개 JSON/API 확인 여부: 확인 안 됨

### 확인한 구조

메인 페이지에서 확인 가능한 정보:

- 축제명
- 행사 기간
- 장소
- 팝업존/공지사항
- 축제개요, 축제일정, 축제프로그램, 오시는 길 링크

확인된 URL 구조:

```txt
/contents.htm?code=2_1   축제개요
/contents.htm?code=2_2   축제일정
/contents.htm?code=3_1_1 프로그램안내
/nwebnics/wBoard/list.php?code=5_1 공지사항
```

프로그램안내 페이지에서 확인 가능한 정보:

- 프로그램명
- 일시
- 장소
- 설명
- 이미지
- 프로그램 분류

### 필드별 추출 가능성

| 필드 | 가능 여부 | 비고 |
| --- | --- | --- |
| 축제명 | 가능 | 메인/개요에서 확인 가능 |
| 기간 | 가능 | 메인에 날짜 표시 |
| 장소 | 가능 | 메인과 프로그램 페이지에서 확인 가능 |
| 프로그램명 | 가능 | 프로그램안내 HTML에서 확인 가능 |
| 이미지 | 가능 | 페이지 내 이미지가 있으나 사용권한 확인 필요 |
| 상세 링크 | 가능 | `contents.htm?code=...` 구조 |

### 한계

- 군산시간여행축제 단일 축제에 특화된 출처입니다.
- 군산 전체 행사·축제 목록을 대체할 수 없습니다.
- 공개 JSON/API가 확인되지 않았습니다.
- 프로그램 일정은 변경 가능성이 있어, 홈에 확정 일정처럼 과장하면 안 됩니다.

### 추천 사용 방식

- `군산시간여행축제` 상세 카드의 공식 출처 링크로 사용합니다.
- 프로그램 단위 데이터는 자동 수집하지 않고, 공식 확인 후 관리자 입력 대상으로 둡니다.
- 기간/장소는 공식 출처 기준으로 표시할 수 있지만, “현재 진행 중” 표현은 별도 날짜 검증 후에만 사용합니다.

## TourAPI와 비교

| 기준 | 군산문화관광 | 군산시간여행축제 | TourAPI |
| --- | --- | --- | --- |
| 공식성 | 높음 | 높음 | 높음 |
| 군산 특화 | 높음 | 특정 축제만 높음 | 지역 필터 필요 |
| 공개 API | 확인 안 됨 | 확인 안 됨 | 있음 |
| 이미지 | 상세 HTML에 있음 | HTML에 있음 | 응답 확인 필요 |
| 날짜/장소 | 상세 HTML에 있음 | HTML에 있음 | 응답 확인 필요 |
| 자동 연결 안정성 | 낮음~중간 | 낮음~중간 | 중간~높음 |
| 홈 카드 적합성 | 수동 등록 기준 적합 | 대표 축제 상세 적합 | API 연결 후보 적합 |

## 추천 결론

1. 군산 공식 출처는 우선 신뢰 기준으로 사용합니다.
2. 군산문화관광은 행사명, 장소, 기간, 이미지, 상세 링크를 확인할 수 있으나 공개 API는 확인되지 않았습니다.
3. 자동 수집보다는 수동 등록 또는 관리자 입력 구조가 먼저 필요합니다.
4. TourAPI는 자동 연결 후보로 유지합니다.
5. 전국문화축제표준데이터는 대표 축제 검증용 백업 데이터로 사용합니다.
