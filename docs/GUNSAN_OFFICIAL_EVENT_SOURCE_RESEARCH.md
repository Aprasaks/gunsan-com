# 군산 공식 행사·축제 출처 운영 기준

## 목적

군산 공식 출처를 행사 목록 자동화에 사용하는 것이 아니라, **코스에 포함할 행사의 공식성·기간·장소·변경 가능성을 검증하는 기준**으로 정리합니다.

이 문서에서는 UI, API, 크롤링, 데이터 저장을 구현하지 않습니다.

## 출처 요약

| 공식 출처 | URL | 제공 방식 | 권장 사용 |
| --- | --- | --- | --- |
| 군산문화관광 축제/행사 | https://www.gunsan.go.kr/tour/m2101 | HTML 목록/상세 | 코스 행사 후보 수동 검증 |
| 군산시청 공지/행사 | 군산시 공식 도메인 | HTML 게시판 | 변경·취소 공지 보조 확인 |
| 군산시간여행축제 | https://festival.gunsan.go.kr | HTML 페이지/게시판 | 시간여행축제 상세 검증 |
| TourAPI | 공공데이터포털 | API | 후보 발견과 필드 보조 확인 |

## 출처 우선순위

1. 행사 주최/주관 공식 페이지
2. 군산시 또는 군산문화관광 공식 페이지
3. 한국관광공사 TourAPI
4. 전국문화축제표준데이터
5. 기타 출처는 후보 발견에만 사용하고 공식 정보로 확정하지 않음

## 군산문화관광

- 공개 JSON/API는 확인되지 않았습니다.
- 목록과 상세에서 행사명, 장소, 기간, 전화번호, 링크 등을 확인할 수 있습니다.
- HTML 구조는 변경될 수 있으므로 자동 연결의 안정성이 낮습니다.
- 이미지가 보여도 재사용 권한을 별도로 확인해야 합니다.

권장 운영:

1. 운영자가 Course에 넣을 행사 후보를 확인합니다.
2. 상세 페이지에서 날짜와 장소를 검증합니다.
3. 공식 URL과 마지막 확인일을 저장합니다.
4. 기존 CourseStop을 추가/대체하는지 기록합니다.
5. 주차와 이동 영향, 취소 확인 방법을 작성합니다.

자동 HTML 파싱은 구현하지 않습니다. 자동화가 필요하면 공식 API나 데이터 제공 협의를 먼저 검토합니다.

## 군산시청 공지

군산시 공식 공지는 행사 변경, 교통 통제, 취소 공지의 보조 출처가 될 수 있습니다.

- 행사 목록을 자동으로 만드는 1순위 출처가 아닙니다.
- 공식 행사 페이지와 내용이 충돌하면 더 최근의 공식 공지를 확인합니다.
- 게시일과 행사일을 혼동하지 않습니다.
- 공지 본문을 복사하지 않고 필요한 사실만 요약합니다.

## 군산시간여행축제 공식 사이트

단일 대표 축제의 기간, 장소, 프로그램, 공지를 확인하는 출처입니다.

코스 활용:

- 근대거리 Course의 기간 한정 변형
- 일반 정류장을 공식 프로그램으로 대체
- 행사일의 보행 동선과 주차 주의사항 추가
- 종료 후 일반 근대거리 Course로 복귀

한계:

- 군산 전체 행사를 대체하지 않습니다.
- 프로그램은 변경될 수 있습니다.
- 공개 API가 확인되지 않았습니다.
- 이미지 재사용 권한은 별도 확인이 필요합니다.

## Course 연결 검증표

행사를 Course에 넣기 전에 다음을 확인합니다.

| 항목 | 확인 내용 |
| --- | --- |
| 공식성 | 주최/주관 또는 군산 공식 출처인가 |
| 기간 | 시작일과 종료일이 현재 공식 정보와 일치하는가 |
| 장소 | Course 동선 안에 실제로 포함 가능한가 |
| 운영 | 시간, 예약, 입장 조건이 확인되었는가 |
| 이동 | 교통 통제와 주차 영향을 확인했는가 |
| 대체 | 취소/혼잡 시 대체 CourseStop이 있는가 |
| 출처 | 공식 URL을 저장했는가 |
| 확인일 | 마지막 확인 날짜가 있는가 |
| 권한 | 이미지와 본문 사용 권한이 있는가 |

## 정보 상태

- `공식 출처 확인`: 공식 페이지의 사실과 링크 확인
- `군산.com 편집 추천`: Course에 넣을 가치와 동선을 편집 판단
- `확인 필요`: 날짜, 운영, 취소 여부의 최신성 불충분
- `현장 차이 제보`: 공식 정보와 실제 운영 차이 제보

공식 출처 확인이 곧 군산.com 추천을 의미하지 않습니다. 추천도 공식 인증처럼 표현하지 않습니다.

## 금지

- 공식 HTML 무단 크롤링
- 공식 본문과 포스터 복제
- 오래된 행사 페이지를 현재 행사로 표시
- 게시일을 개최일로 사용
- 공식 일정 확인 없이 “진행 중” 표시
- 행사 수를 늘리기 위해 Course와 무관한 행사 노출
- TourAPI 응답만으로 자동 Course 편성

## 권장 데이터 메모

향후 타입 작업에서 검토할 의미:

- 공식 출처명
- 공식 상세 URL
- 시작일/종료일
- 장소
- 마지막 확인일
- 확인 상태
- 연결 Course
- 추가/대체 CourseStop
- 이동/주차 메모
- 취소 시 fallback

실제 타입과 데이터 파일은 별도 이슈에서 구현합니다.

## 결론

- 군산 공식 출처는 자동 콘텐츠 공급원이 아니라 검증 기준입니다.
- 행사 콘텐츠는 목록보다 Course와의 관계를 우선합니다.
- 군산문화관광과 축제 공식 사이트는 수동 확인에 적합합니다.
- TourAPI는 후보 발견과 교차 검증 수단입니다.
- 공식 링크와 마지막 확인일을 함께 관리합니다.
- 코스에 도움이 되지 않거나 최신성을 확인할 수 없는 행사는 노출하지 않습니다.
