# 군산 행사·축제 API 후보 점검

## 목적

이 문서는 행사 목록을 메인에 대량 노출하기 위한 조사가 아닙니다. **여행 날짜와 코스에 포함할 수 있는 공식 행사인지 검증하는 보조 출처**를 정리합니다.

이번 문서에서는 API 연결, 키 설정, 크롤링, UI 반영을 하지 않습니다.

## 제품 관점

행사는 Course의 선택적 변형입니다.

예:

- 근대거리 코스의 정류장을 공식 축제 프로그램으로 대체
- 행사 개최일에 주차가 어려워지는 구간 안내
- 비 오는 날 코스에 공식 실내 전시 추가

행사가 있다는 이유만으로 메인 카드에 노출하지 않습니다. 코스와 일정에 실제 도움이 되는지 편집 판단이 필요합니다.

## 후보 요약

| 후보 | 판단 | 코스에서의 역할 |
| --- | --- | --- |
| 한국관광공사 국문 관광정보 서비스 GW | 키 발급 후 응답 확인 | 군산 행사 후보와 공식 기본 정보 확인 |
| 전국문화축제표준데이터 | 대표 축제 보조 검증 | 정기 축제 기간/장소 교차 확인 |
| 군산문화관광 공식 사이트 | 우선 공식 출처 | 수동 검증과 공식 링크 저장 |
| 군산시간여행축제 공식 사이트 | 단일 대표 축제 상세 출처 | 시간여행축제를 코스에 넣을 때 일정 확인 |

## Course 연결에 필요한 최소 정보

- 행사명
- 시작일
- 종료일
- 장소
- 공식 상세 URL
- 공식 출처명
- 마지막 확인일
- 연결할 Course
- 추가 또는 대체할 CourseStop
- 권장 방문 시간
- 이동/주차 주의사항
- 우천/취소 확인 방법

이미지는 필수가 아닙니다. 제공 이미지의 사용권한을 확인하지 못하면 링크만 사용합니다.

## 1. 한국관광공사 국문 관광정보 서비스 GW

- 제공기관: 한국관광공사
- 공공데이터포털: https://www.data.go.kr/data/15101578/openapi.do
- 환경변수 후보: `TOUR_API_SERVICE_KEY`
- 포맷: JSON/XML

확인할 범위:

- 전북특별자치도 `areaCode`
- 군산시 `sigunguCode`
- 행사 조회 날짜 파라미터
- 행사 식별자와 상세 조회
- 공식 상세 링크 제공 여부
- 수정/갱신 시점 확인 가능성

사용 기준:

- API 결과를 자동으로 코스에 넣지 않습니다.
- 군산 필터 결과를 운영자가 공식 출처와 교차 확인합니다.
- 외부 설명문은 그대로 복사하지 않고 코스 맥락으로 재작성합니다.
- 이미지 URL은 권한 확인 전 사용하지 않습니다.
- 데이터가 없거나 실패해도 가짜 seed 행사를 만들지 않습니다.

연결 전 별도 작업:

1. 키 발급과 환경변수 기준 정리
2. 군산 지역코드 확인
3. 응답 샘플 구조 검증
4. Course 연결에 필요한 필드 존재 여부 확인
5. 실패/fallback 설계

## 2. 전국문화축제표준데이터

- 공공데이터포털: https://www.data.go.kr/data/15013104/standard.do
- 포맷: CSV 또는 별도 OpenAPI 변환
- 갱신: 분기 기준으로 알려져 있어 최신성 확인 필요

적합한 용도:

- 대표 정기 축제의 기간과 장소 교차 확인
- 주최/주관기관과 공식 홈페이지 확인
- TourAPI 결과 보조 검증

한계:

- 오늘/이번 주 행사 상태에 부적합
- 작은 공연·전시·체험을 포괄하지 않을 수 있음
- 이미지 중심 UI를 위한 데이터가 아님
- 분기 갱신이므로 “진행 중” 계산에 단독 사용 금지

## 3. 군산문화관광 공식 사이트

- 예시 URL: https://www.gunsan.go.kr/tour/m2101
- 공개 API: 확인되지 않음
- 구조: HTML 목록과 상세 페이지

사용 방향:

- 공식 출처 링크와 날짜/장소를 운영자가 수동 확인
- Course에 넣을 행사 후보의 1차 공식 검증
- 자동 HTML 파싱과 무단 크롤링은 보류
- 공식 제공 협의 또는 API를 크롤링보다 먼저 검토

공식 사이트의 문장과 이미지를 그대로 가져오지 않습니다. 군산.com은 행사 자체를 홍보하는 대신 코스에 넣을 이유와 이동 영향을 편집합니다.

## 4. 군산시간여행축제 공식 사이트

- URL: https://festival.gunsan.go.kr
- 공개 API: 확인되지 않음
- 범위: 군산시간여행축제 단일 행사

사용 방향:

- 시간여행축제를 근대거리 코스에 포함할 때 공식 기간/장소 확인
- 프로그램 단위 정보는 운영자가 공식 출처를 다시 확인
- 일정 변경 가능성을 고려해 마지막 확인일 기록
- 군산 전체 행사 목록을 대체하지 않음

## 검증 흐름

```txt
행사 후보 발견
→ 군산 공식 출처 확인
→ 날짜·장소·취소 공지 확인
→ 기존 Course와의 관계 판단
→ 추가/대체 CourseStop 결정
→ 이동·주차 메모 작성
→ 출처 URL과 마지막 확인일 기록
```

## 표현 원칙

금지:

- 실제 검증 없는 “오늘 진행 중”
- 데이터 근거 없는 “실시간 인기”와 조회수
- 분기 데이터만으로 현재 개최 확정
- 비공식 포스터와 사진 사용
- API 문장을 그대로 상세 설명으로 복사

허용:

- 공식 출처 기준 행사 기간
- 마지막 확인일
- 코스에 포함할 이유
- 기존 정류장과의 대체 관계
- 행사일의 주차/이동 주의사항

## fallback

- 확인되지 않은 행사는 코스에서 제외합니다.
- API 실패 시 공식 링크를 통한 최종 확인 행동을 제공합니다.
- 날짜를 모르면 날짜를 만들지 않습니다.
- 행사 취소/변경을 확인하기 어려우면 `확인 필요`로 표시합니다.
- 대체 CourseStop을 함께 준비합니다.
