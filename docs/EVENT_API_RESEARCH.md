# 군산 행사·축제 API 후보 점검

## 목적

이 문서는 군산.com 메인 홈의 `군산 행사·축제` 섹션에 실제 데이터를 연결하기 전에 API 후보를 점검한 기록입니다.

이번 작업에서는 홈 화면에 실제 API 데이터를 반영하지 않습니다. API 키를 코드에 넣지 않고, `.env` 파일도 수정하지 않습니다.

## 결론 요약

| 후보 | 결론 | 이유 |
| --- | --- | --- |
| 한국관광공사 국문 관광정보 서비스 GW | 키 발급 후 우선 검토 | 행사정보, 이미지정보, 지역 기반 관광정보를 제공하고 JSON/XML을 지원합니다. |
| 전국문화축제표준데이터 | 보조 데이터로 검토 | 정기·대표 축제 성격에 적합하지만 분기 갱신이고 이미지 제공은 기대하기 어렵습니다. |
| 군산문화관광 공식 사이트 | 직접 API 확인 안 됨 | 공식 행사 페이지는 있으나 공개 API는 확인하지 못했습니다. 크롤링 대상이 아닙니다. |

## 1. 한국관광공사 국문 관광정보 서비스 GW

- 제공기관: 한국관광공사
- 공공데이터포털: https://www.data.go.kr/data/15101578/openapi.do
- 신청 필요 여부: 필요
- 필요한 환경변수: `TOUR_API_SERVICE_KEY`
- 데이터 포맷: JSON, XML
- 업데이트 주기: 실시간
- 비용: 무료

### 확인한 제공 범위

공공데이터포털 설명 기준으로 이 API는 지역코드정보, 지역기반관광정보, 위치기반관광정보, 키워드검색, 행사정보, 숙박정보, 공통정보, 소개정보, 반복정보, 이미지정보 등을 제공합니다.

따라서 군산.com에서는 다음 확장에 적합합니다.

- 행사·축제 카드
- 관광/여행 카테고리
- 대표 장소 상세 보강
- 지도에서 보기
- 향후 숙박 정보

### 군산 필터링 가능성

연결 전 확인해야 할 값:

- 전북특별자치도 `areaCode`
- 군산시 `sigunguCode`
- 행사 조회용 날짜 파라미터
- 행사/축제 콘텐츠 타입 또는 행사 전용 엔드포인트

권장 확인 순서:

1. 지역코드 조회로 전북특별자치도와 군산시 코드를 확인합니다.
2. 행사정보 조회에서 해당 `areaCode`, `sigunguCode` 조합으로 군산 데이터가 나오는지 확인합니다.
3. 결과가 부족하면 keyword `군산` 검색을 보조로 검토합니다.
4. `contentId` 기준으로 공통정보, 소개정보, 이미지정보를 추가 조회할 수 있는지 확인합니다.

### 홈 행사 카드에 필요한 필드

다음 필드가 응답에 있는지 테스트해야 합니다.

- 행사명
- 시작일
- 종료일
- 장소 또는 주소
- 대표 이미지 URL
- `contentId`
- 상세 정보 조회에 필요한 타입 또는 식별자

### 사용 시 주의

- 날짜가 있어도 공식 출처 기준으로 확인된 값으로만 표시합니다.
- 이미지 URL은 제공되어도 사용 조건과 저작권 안내를 확인합니다.
- API 실패 시 기존 seed 카드 또는 유형 카드로 fallback합니다.
- 홈 화면에는 3개 정도만 compact하게 노출합니다.
- “오늘 진행 중”, “실시간 인기”, “조회수” 같은 표현은 별도 데이터 없이는 사용하지 않습니다.

### 연결 판단

- 가장 적합한 후보입니다.
- 단, 키 발급 후 실제 응답에서 군산 행사 데이터의 양과 이미지 품질을 먼저 확인해야 합니다.
- 바로 홈 반영하지 말고 `lib/api/events.ts` 초안과 응답 샘플 점검을 별도 작업으로 진행합니다.

## 2. 전국문화축제표준데이터

- 공공데이터포털: https://www.data.go.kr/data/15013104/standard.do
- 제공기관: 한국관광공사, 지방자치단체
- 데이터 포맷: CSV
- 갱신주기: 분기
- 신청 필요 여부: 다운로드 방식이면 키 없이 검토 가능, OpenAPI 자동변환 사용 시 별도 확인 필요

### 확인한 제공 범위

공공데이터포털 설명 기준으로 전국문화축제표준데이터는 매년 정기적으로 개최되는 규모 있는 축제 정보입니다.

공개 항목 예:

- 축제명
- 개최장소
- 축제시작일자
- 축제종료일자
- 축제내용
- 주관기관명
- 주최기관명
- 전화번호
- 홈페이지주소
- 소재지도로명주소
- 소재지지번주소
- 위도
- 경도

### 군산 필터링 가능성

CSV 데이터 안에서 다음 기준으로 필터링 가능성을 확인합니다.

- 소재지도로명주소에 `군산`
- 소재지지번주소에 `군산`
- 주관기관명 또는 주최기관명에 `군산`
- 축제명에 `군산`

### 홈 행사 카드 적합성

적합한 부분:

- 군산 대표 축제 목록
- 정기 축제 정보
- 행사 상세 페이지의 공식 출처 보조 정보

한계:

- 실시간 행사 목록에는 부적합합니다.
- 이미지 URL 제공을 기대하기 어렵습니다.
- 갱신주기가 분기라 오늘/이번 주 행사처럼 표현하면 안 됩니다.
- 공연, 전시, 체험 같은 작은 행사까지 포괄하지 못할 수 있습니다.

### 연결 판단

- TourAPI의 보조 데이터로 적합합니다.
- 군산 대표 축제 seed를 검증하거나, 공식 대표 축제 목록을 만드는 데 먼저 사용합니다.
- 홈 카드의 이미지형 UI를 채우기 위한 1순위 API는 아닙니다.

## 3. 군산문화관광 공식 사이트

- 공식 사이트 예: https://www.gunsan.go.kr/tour/m2101
- 확인 결과: 공개 API 확인 안 됨

군산문화관광 사이트에는 `축제/행사 정보` 메뉴가 있습니다. 다만 이번 점검 범위에서는 공개 API를 확인하지 못했습니다.

### 사용 방향

- 공식 출처 확인용 링크 또는 수동 검증 기준으로 활용합니다.
- 크롤링 구현 대상이 아닙니다.
- 페이지 HTML을 파싱해서 홈 화면에 자동 반영하지 않습니다.

### 연결 판단

- 직접 API 후보는 아닙니다.
- 행사 상세 정보 검증 시 공식 링크로 연결하거나 운영자가 수동 확인할 출처로 둡니다.

## 추천 연결 순서

1. `TOUR_API_SERVICE_KEY` 발급 후 TourAPI 행사 조회 응답 샘플 확인
2. 지역코드 조회로 전북특별자치도/군산시 코드 확정
3. 군산 행사 데이터 개수, 이미지 URL, 날짜, 장소 필드 확인
4. `lib/api/events.ts`에 실패 fallback 포함한 조회 함수 초안 작성
5. 홈 화면 반영 전, seed 카드와 API 데이터의 매핑 기준 정리
6. 전국문화축제표준데이터로 대표 축제 목록 보조 검증

## fallback 전략

API 키가 없거나 응답이 부족한 경우:

- 홈 `군산 행사·축제` 섹션은 현재 seed 카드 유지
- “오늘 진행 중”, “이번 주 행사” 같은 표현 사용 금지
- 실제 날짜를 모르면 날짜 표시 생략
- 이미지가 없으면 기존 중립 썸네일 유지
